# Phish-Intel AI

**Phish-Intel AI** is a hybrid AI & cybersecurity platform designed to detect phishing and malicious URLs in real time.  
It combines:

- **Machine Learning:** Classifies emails and messages using NLP and feature-based models.  
- **Rule-Based Security Engine:** Detects suspicious URLs, domains, and patterns using cybersecurity intelligence.  
- **Explainable Results:** Provides risk scores, classifications, and detailed reasoning for each alert.  

This project showcases **AI deployment, secure backend design, threat modeling, and full-stack integration**, making it an impressive portfolio centerpiece.
